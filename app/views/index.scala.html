Jasminを使ってSwiftを学ぼう

今回の実験にご協力いただき、ありがとうございます。

私たちは現在、変換による新しいプログラミング言語の学び方について研究しています。

既存の勉強方法に変換を取り入れ、常用言語と新しい言語を比較することで新しいプログラミング言語の勉強をサポートしています。

その実現として、変換を取り入れた学習環境をWebアプリとして提供しています。

この実験では、学習環境を使って、Swiftの勉強を取り組んでいただきます。
勉強を始める時の開始時間は別途記録してください。
これからWebアプリの使い方について説明します。

このアプリを使うにChromeをお勧めします。

最初の部分は以下のようになっています。まずはいくつかの選択問題を解いていただきます。

下にはJavaプログラムをSwiftプログラムに変換できる変換器があります。
これを使うには、入力がひとつのJavaファイルの文法に従わなければなりません。
つまり、クラスの定義から始まって、フィールドとメソッドの定義が中にあって、さらにメソッドの中にステートメントがあるということです。

例えば：
class A {
    for (int i = 0; i < 5; i++) {
    }
}
のようなコードはエラーになり、正確に変換できません。 
class A {
    void a() {
        for (int i = 0; i < 5; i++) {
        }
    }
}

これが正しく変換できるJavaプログラムです。

その後にいくつかのコーディングタスクを解いていただきます。

MacユーザーはXCodeを開き、File->New->Playgroundで.playgroundファイルを生成し、Swiftコードを書くことができます。

他のユーザーはオンラインコンパイラーである http://www.runswiftlang.com または http://swiftstub.com/ などを使えます。

ネットを使って参考資料を探すこともできます。

問題を解き終わったら、終了時刻をまた別途記録してください。
そのあと、Webページに提示されたサイトに接続し、テストの受験とアンケートをお願いします。
